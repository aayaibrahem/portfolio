# Aya Ibrahem Kamel Portfolio

A professional and creative portfolio website built with React.js, Tailwind CSS, and Framer Motion.

## Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Dark Mode**: Smooth transition between light and dark themes.
- **Interactive Gallery**: Showcase for Canva designs, CVs, PPTs, and AI videos.
- **Smooth Animations**: Powered by Framer Motion for a modern feel.
- **Project Showcase**: Highlights key development projects with links.

## How to Replace Placeholders
To make this portfolio your own, follow these steps:

1. **Portfolio Assets**:
   - Place your design images in `public/portfolio/designs/`
   - Place your CV PDF in `public/portfolio/cv/` and name it `aya_cv_placeholder.pdf` (or update the link in `src/data/portfolio.ts`)
   - Place your PPT files in `public/portfolio/ppt/`
   - Place your AI videos in `public/portfolio/videos/`

2. **Data Customization**:
   - Open `src/data/portfolio.ts` and update the information, links, and asset paths.

3. **Images**:
   - Replace the `picsum.photos` URLs in `src/data/portfolio.ts` and components with your actual project screenshots and profile picture.
